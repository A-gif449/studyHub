🔔 Notification System

A live demo of the StudyHub notification bell. Try the buttons below to simulate adding PDFs — exactly what happens when you upload from the Admin Panel.

How to add it to your project
Step 01

Copy the file

Place notifications.js in the same folder as your index.html, admin.html, chat.html, and viewer.html.

Step 02

Add the script tag

Paste one line just before </body> in every HTML page. It auto-injects the bell into the existing <nav>.

Step 03

Done — zero config

The bell reads from your Firestore pdfs collection in real-time. Unread counts reset when users open the panel.

Add to each HTML page (just before </body>) 
<!-- StudyHub Notification System -->
<!-- Must come AFTER Firebase SDKs + FIREBASE_CONFIG -->
<script src="notifications.js"></script>
What's included

Real-time counter — listens to Firestore live. Badge appears instantly when you upload from Admin Panel, no page refresh needed.

Persistent unread state — uses localStorage so the count survives page reloads and navigation between pages.

Clickable items — each notification goes straight to viewer.html?id=…. Opening the panel marks everything as read.

Bell animation — the bell shakes when new PDFs arrive. Badge scales in with a spring animation and shows 99+ when needed.

Zero config, auto-injects — finds .nav-right automatically. Works in all 4 pages (index, admin, chat, viewer) with one script tag each.
