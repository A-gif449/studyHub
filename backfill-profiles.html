🔧 Backfill User Profiles

One-time admin tool. Scans views, downloads, follows, friendRequests, and friendships to find every unique user this app has ever seen, then creates a userProfiles/{uid} doc for anyone missing one — so every user resolves correctly on profile.html, even if they've never viewed/downloaded a PDF.

Admin login required. Safe to run multiple times — existing profiles are never overwritten, only missing ones are created.
